# MyndAIX Agent Toolkit

Open-source tools from [MyndAIX](https://myndaix.com) — an AI lab that builds production software with autonomous agent teams.

## Tools

### [Voice Hotkey](./voice-hotkey/)
System-wide push-to-talk for macOS. Hold Right Command, speak, text appears wherever your cursor is. Local Whisper transcription — no cloud, no API keys, instant.

### [Auto-Review Hook](./auto-review-hook/)
Claude Code hook that triggers Codex to review every file you edit. Different model family catches different blindspots. Diff-aware, non-blocking, runs in background.

## Philosophy

At MyndAIX, we run a team of 8 AI agents across 3 model families (Claude, Codex, Gemini). These tools are extracted from our daily workflow — battle-tested, not theoretical.

**Claude builds. Codex reviews. Gemini provides the third perspective.**

That's not redundancy. That's coverage.
